## 🎉 使用

### 指令表

| 指令 | 权限 | 需要@ | 范围 | 说明 |
|:-----:|:----:|:----:|:----:|:----:|
| /车票 【出发站】 【到达站】 【日期（选填）】 | 群员 | 否 | 群聊、私聊 | 查询从出发站至到达站间的列车票价及余票数量 |

**其中，选填的【日期】可以填以下几种格式，如不填日期或所填日期小于今天，一律默认为今天**
- `今天`、`明天`
- `2026-03-12`
- `2026年3月12日`
